**目的: CSSプロパティで指定された画像URLを取り出したい**
<code>background-image: url( https://xxxxx.png );</code>

```php
<!--?php-->
function cutOutUrl($url)
{
  preg_match(&#39;/(https).*\.(png|jpg|jpeg)/i&#39;, $url, $match);
  return $match[0] ?? null;
}

$url = cutOutUrl("background-image: url( https://xxxxx.png );");
echo $url;

```

$ php index.php
